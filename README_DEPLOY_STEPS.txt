CRICKET CAREER - ADSENSE-READY GITHUB PAGES PACKAGE

What is included:
- Homepage with real written content
- Playable cricket prototype
- Cinematic toss, player entry, field huddle, live camera, batting, bowling and highlights
- AdSense-friendly content pages
- Privacy, Terms, Contact and Disclaimer pages
- ads.txt, robots.txt and sitemap.xml

Important before upload:
1. Replace the placeholder email in contact.html and privacy.html with your public contact email.
2. Keep real cricketer photos, official logos, official team badges and copied commentary out of the site unless you have permission.
3. Keep AdSense manual ad units away from the game canvas, loading screens, blank screens and alert-only screens.
4. This package includes the AdSense script on content pages, but not on play.html.
5. Upload all files to the root of diliptiwari.github.io repo, not inside a subfolder.

GitHub upload steps:
1. Open your GitHub repository: DilipTiwari/diliptiwari.github.io or your GitHub Pages root repo.
2. Click Add file > Upload files.
3. Drag all files and folders from this package.
4. Commit changes.
5. Wait 2-5 minutes.
6. Test these URLs:
   https://diliptiwari.github.io/
   https://diliptiwari.github.io/play.html
   https://diliptiwari.github.io/ads.txt
   https://diliptiwari.github.io/sitemap.xml
7. After all pages work, go to AdSense and request review again.
