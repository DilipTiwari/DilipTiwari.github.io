About Cricket Career

Cricket Career is an independent browser-based cricket game concept. The goal is to create a lightweight cricket experience that works on mobile, desktop and static hosting such as GitHub Pages.

The game is focused on cinematic presentation: toss ceremony, pitch camera, player walk-in, fielding huddle, batting shots, bowling choices, umpire actions and match highlights.

Independent project

This project is not affiliated with any cricket board, tournament, league, player, broadcaster or sponsor. All rosters in the prototype are fictional and used only for gameplay demonstration.

Why web first?

A web version is easy to share, test and improve. It also allows quick updates without asking users to install an app.
